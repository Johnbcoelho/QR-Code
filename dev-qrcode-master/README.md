## Visão Geral

Projeto de Gerador de QR Code: Crie facilmente seus próprios QR Codes para compartilhar e divulgar informações de maneira rápida e eficiente. Ideal para promoções, cartões de visita digitais, links para redes sociais, websites, entre outros. A aplicação permite gerar QR Codes personalizados que podem ser integrados às suas campanhas e estratégias de marketing.

## Começando

### Pré-requisitos

- Node.js (v14.x ou superior)
- npm

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/DalioSY/dev-qrcode
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd dev-qrcode
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```
